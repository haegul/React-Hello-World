# Lecture
## Codepen.io 사용하기
[Codepen.io](https://codepen.io/pen/)에 접속하여 ES6 & React 환경 설정을 해준다.  
#### ES6
Setting > Javscript > Javascript Preprocessor  
Babel 선택

#### React
Setting > Javscript > Quick-add  
React, React-DOM 선택(위 아래 2개에 따로 선택해서 넣으면 됩니다.)
